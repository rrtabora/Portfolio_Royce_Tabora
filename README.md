# Portfolio_Royce_Tabora
This git serves as a space for my projects, works, practices, and serves as my portfolio. It highlights modern end-to-end data engineering and analytics portfolio project built using AWS, Snowflake, dbt, Power BI, Snowflake Cortex, Streamlit, and external AI integrations.

The project demonstrates how raw operational data can be ingested, transformed, modelled, governed, analysed, and exposed through business intelligence dashboards and AI-powered applications.

---
## Project Status

- Secure AWS-to-Snowflake integration
- Raw data ingestion into Snowflake

- Dimensional data modelling
- Fact and dimension tables
- Data marts for business reporting
- Data quality tests and documentation
- Customer review sentiment analysis
